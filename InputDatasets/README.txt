This folder contains the input datasets which were used to produce the study questions.

=====================================
AttentionTask Folder

This folder contains the datasets which where used to produce the Attention task, for both the AccuStripes experiment and the validation study.

=====================================
Questions_AccuStripesStudy Folder

This folder contains the datasets which where used to produce the study questions for the AccuStripes experiment.
The datasets are ordered according to the composition for which they were used.

=====================================
Questions_ValidationStudy Folder

This folder contains the datasets which where used to produce the study questions for the validation study.
